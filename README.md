# Install
clone the repository
```
$ git clone https://github.com/carlos4rias/eje-turismo.git
```
install the dependencies

```
$ yarn install
```

# Run an instance
+ Be sure your mongodb server is running
+ Create a root user if you have not
```
npm init
```
+ just type
```
$ npm start
```
